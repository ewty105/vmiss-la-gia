## 一、CN2 GIA 到底是条什么线，凭什么贵

VMISS 洛杉矶 CN2 GIA 系列的精妙之处在于"回程强制三网 GIA"。翻译成人话就是：

- **电信用户**：去程走 CN2 GIA，回程也走 CN2 GIA，双向 VIP 待遇，体验最爽。
